[441. Arranging Coins](https://leetcode.com/problems/arranging-coins/)

* GoDaddy
* Math, Binary Search

## Method 1. Binary Search
```java 
class Solution {
    public int arrangeCoins(int n) {
        long left = 0;
        long right = n;
        long mid = 0;
        long sum = 0;
        while(left <= right) {
            mid = left + (right - left) / 2;
            sum = mid * (mid + 1) / 2;
            
            if(sum == n) {
                return (int)mid;
            }
            
            if(n < sum) {
                right = mid - 1;
            } else {    // n >= sum
                left = mid + 1;
            }
        }
        return (int)right;
    }
}
```
