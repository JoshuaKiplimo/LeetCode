## Method 1. 
For `num > 2`, the square root a is always less than num/2 and greater than 1: 1 < x < num/2.
```java 
class Solution {
    public boolean isPerfectSquare(int num) {
        if(num < 2) {   // 0 and 1
            return true;
        }
        
        long left = 2;
        long right = num / 2;
        long x, guessSquared;
        while(left <= right) {
            x = left + (right - left) / 2;
            guessSquared = x * x;
            if(guessSquared == num) {
                return true;
            } else if(guessSquared > num) {
                right = x - 1;
            } else {
                left = x + 1;
            }
        }
        return false;
    }
}
```


## Method 2. Newton's Method
```java 
class Solution {
    public boolean isPerfectSquare(int num) {
        if(num < 2) {
            return true;
        }
        
        long x = num / 2;
        while(x * x > num) {
            x = (x + num / x) / 2;
        }
        return x * x == num;
    }
}
```
