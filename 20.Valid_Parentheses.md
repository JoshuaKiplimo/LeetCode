## Method 1. Stack
```java 
class Solution {
    public boolean isValid(String s) {
        if(s == null || s.length() == 0) {
            return true;
        }
        
        if(s.length() % 2 == 1) {
            return false;
        }
        
        Stack<Character> stack = new Stack<Character>();
        for(char c: s.toCharArray()) {
            switch(c) {
                case '(':
                case '[':
                case '{':
                    stack.push(c);
                    break;
                case ')':
                    if(stack.isEmpty() || stack.pop()!='(') {
                        return false;
                    }
                    break;
                case ']':
                    if(stack.isEmpty() || stack.pop()!='[') {
                        return false;
                    }
                    break;
                case '}':
                    if(stack.isEmpty() || stack.pop()!='{') {
                        return false;
                    }
                    break;
            }
        }
        return stack.isEmpty();
    }
}
```

Complexity:
    
    Time: O(N)
    Space: O(N)
