## Method 1. Binary Search
```java
class Solution {
    public int mySqrt(int x) {
        if(x < 2) {
            return x;
        }
        
        long lo = 1;
        long hi = x;
        long mid;
        long square;
        while(lo <= hi) {
            mid = lo + (hi - lo) / 2;
            square = mid * mid;
            if(square == x) {
                return (int)mid;
            } else if(square < x) {
                lo = mid + 1;
            } else {
                hi = mid - 1;
            }
        }
        return (int)hi;
    }
}
```
